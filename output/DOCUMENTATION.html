Source Data
البيانات المصدرية

Two source files. Zero shared keys.

ملفّان مصدريان. بلا مفتاح ربط مشترك.

processed-products.xlsx — 147 rows of Arabic titles, categories, prices, and MPNs. Semi-structured with HTML tags in descriptions and service rows mixed with physical products.

processed-products.xlsx — ١٤٧ صفاً من العناوين العربية والفئات والأسعار وأرقام المصنّعين. شبه منظّم مع وسوم HTML في الأوصاف وصفوف خدمات مختلطة بالمنتجات الفعلية.

urls.txt — Raw image URLs with no mapping to products. Model numbers buried in URL slugs were the only matching signal.

urls.txt — روابط صور خام بدون أي ربط بالمنتجات. أرقام الطُّرز المدفونة في مسارات الروابط كانت إشارة المطابقة الوحيدة.

  • No shared key between product data and image URLs
  • Duplicated products at different price tiers
  • Service rows mixed with physical products
  • Raw HTML tags embedded in descriptions
  • Inconsistent Arabic/English hybrid titles
  • لا يوجد مفتاح مشترك بين بيانات المنتجات وروابط الصور
  • منتجات مكرّرة بمستويات أسعار مختلفة
  • صفوف خدمات مختلطة مع المنتجات الفعلية
  • وسوم HTML خام مضمّنة في الأوصاف
  • عناوين هجينة عربية-إنجليزية غير متّسقة

05
Data Rules
قواعد البيانات

Seven rules that shaped every row

سبع قواعد صاغت كل صف

Rule 01
القاعدة ٠١
Strict column order
ترتيب أعمدة صارم
All output files follow the exact 14-column order specified by Google's Product Data Specification.
جميع ملفات الإخراج تتبع ترتيب الأعمدة الأربعة عشر المحدد في مواصفات بيانات منتجات جوجل بالضبط.
Rule 02
القاعدة ٠٢
One image per product
صورة واحدة لكل منتج
Only the single best-matching URL is kept in image_link. Extras discarded to avoid GMC errors.
يُحتفظ فقط بالرابط الأفضل مطابقةً في حقل image_link. تُرفض الروابط الزائدة تجنباً لأخطاء مركز التجّار.
Rule 03
القاعدة ٠٣
VAT-inclusive SAR pricing
أسعار بالريال شاملة الضريبة
All prices formatted as XXXX.XX SAR. Saudi 15% VAT is already baked into source prices.
جميع الأسعار بصيغة XXXX.XX ريال سعودي. ضريبة القيمة المضافة ١٥٪ مضمّنة أصلاً في أسعار المصدر.
Rule 04
القاعدة ٠٤
Availability mapping
ربط حالة الإتاحة
Arabic متاح / غير متاح converted to GMC-standard in_stock / out_of_stock values.
تحويل القيم العربية «متاح» و«غير متاح» إلى القيم القياسية in_stock و out_of_stock.
Rule 05
القاعدة ٠٥
Brand normalization
توحيد العلامات التجارية
Arabic + English brand names like "ZKTeco - زد كي تيكو" simplified to English-only for GMC compliance.
أسماء العلامات التجارية المزدوجة مثل «ZKTeco - زد كي تيكو» تم تبسيطها إلى الإنجليزية فقط لتوافق مركز التجّار.
Rule 06
القاعدة ٠٦
Services excluded
استبعاد الخدمات
8 service/consultation rows removed — GMC requires physical or digital goods, not service inquiries.
حُذفت ٨ صفوف خدمات واستشارات — يشترط مركز التجّار منتجات مادية أو رقمية وليس طلبات خدمة.
Rule 07
القاعدة ٠٧
No GTIN, MPN used
بدون GTIN، استُخدم MPN
Products lack universal barcodes, so gtin is empty and identifier_exists is set to "yes" when an MPN is present.
لا تحمل المنتجات باركودات عالمية، لذا حقل gtin فارغ وidentifier_exists مضبوط على «yes» عند وجود رقم المصنّع.

07
Implementation
التنفيذ

How to use each file

كيف تستخدم كل ملف

1

Upload GMC Feed

رفع خلاصة مركز التجّار

Open gmc_product_feed.csv or .xlsx. Upload via Merchant Center → Products → Feeds → Add feed. Select Saudi Arabia as target country and SAR as currency.

افتح gmc_product_feed.csv أو .xlsx. ارفعه عبر مركز التجّار ← المنتجات ← الخلاصات ← إضافة خلاصة. اختر المملكة العربية السعودية كبلد مستهدف والريال السعودي كعملة.

2

Verify Image URLs

تحقق من روابط الصور

Confirm image URLs resolve to img.nahamtech.org. Only 1 product (Zebra USB cable) has no matched image.

تأكد أن روابط الصور تشير إلى img.nahamtech.org. منتج واحد فقط (كابل Zebra USB) بدون صورة مطابقة.

3

Import Ads via Google Ads Editor

استيراد الإعلانات عبر محرّر إعلانات جوجل

Open Google Ads Editor → Account → Import → From CSV. Select google_ads_campaign.csv. Map columns: Ad Group, Keyword, Match Type, Headlines, Descriptions.

افتح محرر إعلانات جوجل ← الحساب ← استيراد ← من CSV. اختر google_ads_campaign.csv. اربط الأعمدة: المجموعة الإعلانية، الكلمة المفتاحية، نوع المطابقة، العناوين، الأوصاف.

4

Use JSON for API Integration

استخدم JSON للتكامل البرمجي

Use the .json files for integration with custom dashboards, automation scripts, or feed management platforms. Structure is a flat array of objects keyed by column name.

استخدم ملفات .json للتكامل مع لوحات المعلومات أو نصوص الأتمتة أو منصات إدارة الخلاصات. البنية مصفوفة مسطّحة من الكائنات مفهرسة بأسماء الأعمدة.
