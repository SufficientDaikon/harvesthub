06 — Quick Start

Three minutes to
first harvest.

No accounts. No API keys. No credit cards.

1

Install Dependencies

Node.js 18+ and Python 3.10+ required.

npm install && pip install scrapling orjson browserforge curl_cffi
2

Add Your URLs

One product URL per line. Any e‑commerce site.

echo https://books.toscrape.com/catalogue/a-light-in-the-attic_1000/index.html > urls.txt
3

Harvest

Run the scraper. Watch it work. Get your data.

npx tsx src/cli/index.ts scrape --urls urls.txt --output products.xlsx
4

Launch Dashboard

Explore, search, filter, and export from the web UI.

npx tsx src/cli/index.ts dashboard --port 4000
