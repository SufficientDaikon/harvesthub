المميزات

كل ما تحتاجه لاستخراج
بيانات المنتجات

من الاستخراج الذكي إلى التصدير الاحترافي، HarvestHub تتولى سلسلة العمل بالكامل حتى تركّز أنت على ما يهم — البيانات.

🔍

استخراج تكيفي

محرك ثلاثي المستويات: JSON-LD ثم البيانات الوصفية ثم التحليل الذكي. كل حقل يحصل على تقييم ثقة.

📊

تصدير احترافي

ملفات Excel مصممة بأناقة، CSV، JSON، وخلاصات Google Merchant Center. جاهزة للعرض فوراً.

🖥️

لوحة تحكم فورية

واجهة SaaS احترافية مع تحديثات لحظية، رسوم بيانية، بحث، وفلترة متقدمة.

⏱️

جدولة تلقائية

جدولة عمليات الاستخراج بتعبيرات Cron. اضبطها وانسَ الأمر — البيانات تبقى محدّثة.

📈

تتبع الأسعار

كشف تلقائي لتغيرات الأسعار بين عمليات الاستخراج مع سجل كامل ومقارنة تاريخية.

🔄

تحكم بالسرعة

محدد معدل ذكي لكل نطاق مع تراجع أُسّي وتصنيف الأخطاء. مرونة أمام الأعطال.

🛡️

تدوير البروكسي

دعم قوائم بروكسي مع تدوير تلقائي وبصمة TLS ووضع التخفي وإدارة الجلسات.

⚡

سطر أوامر قوي

٦ أوامر متكاملة — استخراج، تصدير، حالة، ترحيل، لوحة تحكم، جدولة. بدون إعدادات.

آلية العمل

من الروابط إلى البيانات
في أربع خطوات

لا إعدادات معقدة. لا مفاتيح API. لا اشتراكات مدفوعة. فقط أضف الروابط وشغّل أمراً واحداً.

١

التثبيت

ثبّت المنصة بأمر واحد. لا حاجة لإعداد يدوي.

$ npm install && npm run setup
٢

أضف الروابط

أنشئ ملفاً نصياً بروابط المنتجات — رابط لكل سطر. يدعم أي متجر.

# urls.txt
https://amazon.com/dp/B0CHX3...
https://bestbuy.com/site/...
٣

استخرج البيانات

شغّل أمر الاستخراج. HarvestHub يتولى الاتصالات وإعادة المحاولة والاستخراج تلقائياً.

✓ iPhone 15 Pro — 95% conf
٤

صدّر واستكشف

أمر واحد للتصدير. اختر XLSX أو CSV أو JSON أو خلاصة Google Merchant Center.

$ npx harvest export -f xlsx
البداية السريعة

جاهز للعمل
في أقل من دقيقتين

انسخ، الصق، استخرج. لا ملفات إعداد، لا مفاتيح API، لا تسجيل حسابات. يعمل فوراً.

التثبيت والإعداد 
# Clone and install
$ git clone https://github.com/SufficientDaikon/harvesthub.git
$ cd harvesthub
$ npm install

# Setup Python engine + database
$ npm run setup
  ✓ Python venv created
  ✓ Scrapling installed
  ✓ Database migrated
  ✓ Ready to harvest!
استخراج المنتجات 
# Scrape from a URL list
$ npx harvest scrape --urls urls.txt

التصدير

أربع صيغ،
بدون تنازلات

كل تصدير جاهز للإنتاج فوراً. جداول بيانات مصممة، JSON مهيكل، وخلاصات متوافقة مع Google.

📗
.xlsx

Excel احترافي

تنسيق داكن أنيق مع لوحة ملخص، تصفية تلقائية، روابط قابلة للنقر، وألوان متناوبة. جاهز لأصحاب القرار.

📄
.csv

CSV عالمي

UTF-8 مع BOM لتوافق مثالي مع Excel. عناوين نظيفة، حقول محمية. خفيف وعالمي.

🔷
.json

JSON منظم

غلاف بيانات وصفية مع عدد المنتجات والطوابع الزمنية وإصدار المخطط. مهيكل وجاهز للمطورين.

🛒
.tsv

Google Merchant Center

خلاصة منتجات متوافقة بالكامل مع تحذيرات التحقق. ارفعها مباشرة إلى Google — بدون معالجة إضافية.
