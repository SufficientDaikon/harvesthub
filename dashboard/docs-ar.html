06 — البدء السريع

ثلاث دقائق حتى
أول حصاد.

بدون حسابات. بدون مفاتيح API. بدون بطاقات ائتمان.

1

تثبيت المتطلبات

يتطلب Node.js 18+ و Python 3.10+ أو أحدث.

npm install && pip install scrapling orjson browserforge curl_cffi
2

أضف الروابط

رابط منتج واحد في كل سطر. أي موقع تجارة إلكترونية.

echo https://books.toscrape.com/catalogue/a-light-in-the-attic_1000/index.html > urls.txt
3

ابدأ الحصاد

شغّل المستخرج. راقب العمل. احصل على بياناتك.

npx tsx src/cli/index.ts scrape --urls urls.txt --output products.xlsx
4

شغّل لوحة التحكم

استكشف، ابحث، فلتر، وصدّر من واجهة الويب.

npx tsx src/cli/index.ts dashboard --port 4000
