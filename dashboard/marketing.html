Features

Everything You Need to
Harvest Product Data

From intelligent extraction to premium exports, HarvestHub handles the entire pipeline so you can focus on what matters — the data.

🔍

Adaptive Extraction

3-tier strategy — JSON-LD, Microdata, then CSS heuristics. Every field gets a confidence score so you know what to trust.

🛡️

Anti-Bot Bypass

Powered by Scrapling with TLS fingerprinting, stealth mode, and automatic session management. Scrape sites that block others.

🔄

Smart Retry Engine

Exponential backoff with jitter, error classification, and per-domain rate limiting. Resilient to transient failures.

📊

Premium Exports

Styled XLSX with summary dashboards, CSV with BOM, typed JSON with metadata, and Google Merchant Center feeds.

🖥️

Real-Time Dashboard

WebSocket-powered live updates, interactive charts, search, filters, pagination, and dark/light theme support.

⚡

CLI Power Tools

6 commands — scrape, export, status, migrate, dashboard, schedule. Zero config, one-line operations, instant results.

📈

Price Tracking

Detect price changes between scrape runs. Historical comparison and diff reporting built into the pipeline.

⏱️

Cron Scheduling

Automate recurring scrapes with configurable cron expressions. Set it and forget it — data stays fresh.

Workflow

From URLs to Data
in Four Steps

No complex configuration. No API keys. No paid subscriptions. Just add URLs and run one command.

1

Add Your URLs

Create a text file with product URLs — one per line. Supports any e-commerce site.

# urls.txt
https://amazon.com/dp/B0CHX3...
https://bestbuy.com/site/...
2

Run One Command

Execute the scrape command. HarvestHub handles connections, retries, and extraction automatically.

$ npx harvest scrape --urls urls.txt
3

Automatic Extraction

Each product is extracted via JSON-LD → Microdata → CSS. Every field gets a confidence score.

✓ iPhone 15 Pro — 95% conf
4

Export Anywhere

One command to export. Choose XLSX, CSV, JSON, or Google Merchant Center TSV feed.

$ npx harvest export -f xlsx
Quick Start

Up and Running
in Under 2 Minutes

Copy, paste, scrape. No configuration files, no API keys, no account signup. It just works.

Install & Setup 
# Clone and install
$ git clone https://github.com/SufficientDaikon/harvesthub.git
$ cd harvesthub
$ npm install

# Setup Python engine + database
$ npm run setup
  ✓ Python venv created
  ✓ Scrapling installed
  ✓ Database migrated
  ✓ Ready to harvest!
Scrape Products 
# Scrape from a URL list
$ npx harvest scrape --urls urls.txt

  🌾 HarvestHub Scraper v1.0.0

  ✓ Python engine ready
  ✓ 12 URLs to scrape

  [1/12] ✓ iPhone 15 Pro Max — 95% — 1.2s
  [2/12] ✓ Samsung Galaxy S24 — 88% — 0.9s
  [3/12] ✓ Sony WH-1000XM5 — 92% — 1.1s
Export Data 
# Export to styled Excel workbook
$ npx harvest export -f xlsx -o products.xlsx

  📊 Export Summary
  Format:   XLSX (styled workbook)
  Products: 104
  Sheets:   Summary + Products + Price History

  ✓ Exported to products.xlsx (248 KB)
Launch Dashboard 
# Start the dashboard server
$ npx harvest dashboard
